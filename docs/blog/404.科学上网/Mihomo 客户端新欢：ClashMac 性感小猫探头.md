> 如果你是 Windows 用户，建议先回避一下，不然这篇文章高低让你消费一台 Mac Mini。

自从折腾了 [SubStore 使用 Mihomo 覆写脚本生成订阅](SubStore%20使用%20Mihomo%20覆写脚本生成订阅.md) 以后，又对第三方 Clash GUI 客户端产生了一点点兴趣。Clash Party 用了一年多了，感觉还是有几点不爽：

1. UI 欠缺流畅
2. 入侵配置太多，而且无法完全关闭
3. 占用内存比较高

Clash Party 的上一任——Clash Verge Rev 在上面几点比 Clash Party 更好，可是某些 Bug 一直反反复复。

经过 1024 分钟的横评，终于找到了一款 Mac 平台专供的客户端：探头小猫——**ClashMac**。

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260112014412928.png)

点击菜单栏里探出的猫头，展示出来的菜单非常性感。不过更有新鲜感的在它的管理面板里，为了避免多图杀猫，我做成了视频。

@[youtube](lCQ6rnB1j0g)

## 亮点

最后总结一下这款软件的亮点：

1. 对订阅配置的入侵比较少，不像某 Party 那样自作聪明
2. 内存占用比较理想，隐藏面板的情况下大概 100MB 出头（数据源自 Mac Mini M4 顶丐版）
3. 目前作者处于鸡血状态，更新速度很快，软件的完成度我认为比 Verge 和 Party 更高了

如果受够了 Verge 的 bug 地狱、Party 的臃肿、FlClash 的桌面端二等公民，想找一个入侵配置很少的 GUI 客户端，ClashMac 真的是个不错的选择。
