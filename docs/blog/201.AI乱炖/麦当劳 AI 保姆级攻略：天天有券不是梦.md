### 第一步：注册麦当劳 MCP 平台

首先打开[麦当劳 MCP 平台](https://open.mcd.cn/mcp)，注册一个账号。这可是通往"优惠券自由"的大门！🚪

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260118001519198.png)

### 第二步：激活你的专属 Token

登录后点击"控制台"，激活你的 Token。记住，这串字符就是你的优惠券钥匙！🔑

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260118001223508.png)

复制生成的 Token，然后打开那个有着标志性图标的万能聊天软件——你懂的！😉

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260118011650643.png)

搜索 `@MaiMaiMCPBot`，在聊天窗口中输入 `/token 刚才复制的token`

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260118011850657.png)

当看到 `Token 已保存，可以开始使用指令了。` 的提示时，恭喜你！你已经成功绑定了麦当劳 AI 机器人，接下来就可以坐等优惠券自动上门了！🎉
