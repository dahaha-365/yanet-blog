昨天才提交的申请，今天 iStoreNextD 就拉内测群了。然后晚上吃完晚饭就收到安装镜像了，迫不及待的体验了一番。带大家云尝鲜。

## 安装过程

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260313205813194.png)

由于近期内存比电脑贵，智能用虚拟机先体验一下了。安装界面是经典的 Linux 味道，顺便还做了一波自我介绍。

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260313205948413.png)

安装完成后默认会进了一个控制台 tui，这一点比飞牛和 iStoreOS 都好。如果网络设置错了，在这里还能救。

## 新界面

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260313211529599.png)

第一次登陆 web 界面，需要跟着向导完成初始化设置。

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260313211548774.png)

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260313211603913.png)

不愧是做软路由起家的。先给 iStoreNextD 定一个身份。路由模式就是充当路由器了，交换模式就是充当 NAS 或者旁路由，家庭的第二网关。

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260313211622873.png)

储存路径可以选择单一的默认路径或者按用户划分目录。这个对单身狗是非常友好的，毕竟家里也没有另一个用户👍。

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260313211640564.png)

然后是选择主题。有 Windows 10 和 macOS 两种风格。可以看出 Windows 11 是不得民心的。由于是内测，建议选 macOS，作为没用过苹果电脑的穷人，即使界面有什么 bug 也看不出来🐶。

![](https://wechat-mp0766.oss-cn-chengdu.aliyuncs.com/20260313211726657.png)

嗯，准备掀盖头了。

## 累了，先洗洗睡了

目前只体验了大概 2.5%。总的来说，应该是在 iStoreOS 基础上进行了一次大重构，而且目前的完成度已经非常高了。很多在 iStoreOS 需要插件的功能都已经内置了，飞牛在天上应该会感受到冷吧？不过，目前还缺失了 APP 端，对于习惯了用飞牛备份照片的用户来说，还是非常麻烦了。如果能早点推出 APP 应该就能早点告别 NAS 要装双系统的烦恼了。好了，累了，先洗洗睡了。
